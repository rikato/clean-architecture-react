# Example repo for clean architecture principles in React.

Using MobX, Typescript and React.

```
├───application
│   ├───hooks
│   └───viewModels
├───components
│   ├───general
│   └───molecules
├───core
│   ├───commands
│   ├───domain
│   │   └───entities
│   └───queries
└───infrastructure
    ├───api
    │   ├───common
    │   ├───endpoints
    │   ├───mappers
    │   └───methods
    ├───repositories
    └───services
 ```
